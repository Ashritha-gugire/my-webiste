State-Level Analysis: Education, Incarceration & Public Safety (Re-designing Bad Graphs)

Project Overview: This project analyzes the relationship between state-level education and prison spending alongside public safety outcomes through advanced data visualization and regional mapping across the United States.

  • Research Methodology - Data Collection from multiple public sources, Regional Segmentation, Integration of poverty rates as a key analytical variable
  • Key Findings - Identified significant disparities, Discovered correlations between regional poverty rates and public safety outcomes, Analyzed impact of funding and mapped regional variables
  • Data Visualization Approaches - Interactive US state map showing regional distributions, Comparative analysis charts of education vs. prison funding

Skin Cancer Diagnosis - Advanced Image Processing with PyTorch (Github Link)

Project Overview: This project aims to develop an image processing algorithm using Convolutional Neural Networks (CNNs) to identify specific types of skin cancer from 10,000 clinical images.

  • Technical Approach - PyTorch implementation with custom CNN architecture, Advanced preprocessing techniques for medical imaging, Modified ResNetX101 optimized for skin lesion classification
  • Healthcare Impact - Enhanced capability for early-stage skin cancer identification, Assists medical professionals in diagnostic decision-making, improved accuracy
  • Future Development - Integration of HMM-based genomic data analysis, Expansion of the training dataset for improved accuracy, Development of user-friendly interface for clinical deployment

Predicting Loan Defaults - Machine Learning for Risk Assessment

Project Overview: Developed a comprehensive loan default prediction system using Logistic Regression and Decision Trees to analyze borrower characteristics and loan attributes for accurate risk assessment.

  • Key Insights - Larger loan amounts show lower default rates, suggesting more careful planning, Bankruptcy history increases default risk (41.8% vs 11.5%)
  • Model Development - Implemented logistic regression for primary analysis, Conducted extensive exploratory data analysis, Developed decision tree models for comparison, Evaluated model performance using accuracy metrics Applications & Impact
  • Future Development - Investigate anonymized variables in depth, Experiment with advanced machine learning models, Explore demographic and cultural factors

Virginia Climate Data Dashboard - Climate Adaptation and Resilience Planning Tool (Dashboard Link)

Project Overview: Developed an interactive climate data dashboard for Virginia's municipal planners, integrating NOAA's NCEI data to support evidence-based climate adaptation strategies and resilience planning.

  • Technical Architecture - Python-based data pipeline for NOAA nClimGrid data extraction, County-level geographical data integration, Bankruptcy history increases default risk (41.8% vs 11.5%)
  • Research Insights - 2.53°F average monthly increase statewide, 7.65 inches statewide increase with regional variations, Northward progression of extreme heat challenges, 12 counties showing declining precipitation patterns
  • Practical Applications - Supporting evidence-based climate adaptation strategies, Informing climate resilience initiatives, Guiding water and environmental resource planning
